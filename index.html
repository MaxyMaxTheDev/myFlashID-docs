myFlashID — Developer Documentation

myFlashID is a modern authentication system that lets users log in with a single identity across your apps. Integrate it into your app by redirecting users to the authorization endpoint, then handling the callback.

Authorization Endpoint

GET https://auth.myflashid.com/authorize?client_id=YOUR_CLIENT_ID&redirect_uri=CALLBACK_URL&response_type=code

Token Exchange

POST https://auth.myflashid.com/token
Content-Type: application/json
{
  "client_id": "YOUR_CLIENT_ID",
  "client_secret": "YOUR_CLIENT_SECRET",
  "code": "AUTH_CODE",
  "redirect_uri": "CALLBACK_URL",
  "grant_type": "authorization_code"
}

Callback URL Preview

Redirect will look like:
https://example.com/auth/callback/myflashid/?code=123456
